## 玩法說明
角色操作：WASD or 方向鍵移動
角色攻擊：Ctrl（連按可進行連擊）
角色視角切換：F5
確定鍵、與NPC對話：Z
能力值視窗開啟：F
物品欄視窗開啟：I
技能值視窗開啟：K
技能使用－上排數字1：鎖血技能
技能使用－上排數字2：硬化技能
道具使用－Delete：雜草（恢復50%HP）
道具使用－End：雜花（恢復50%MP）
道具使用－PageDown：中藥（恢復100%HP、MP）

※角色無法後退移動，只能靠旋轉角度往前走。
